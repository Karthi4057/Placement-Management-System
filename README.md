  
The Campus Placement Management System is a MERN-stack web application designed to centralize and automate campus placement activities. It provides secure role-based authentication for admins and students, enabling efficient management of company details, recruitment rounds, and interview questions. Students can register, upload resumes, apply for placement drives, and track their application status in real time. The system also includes admin reporting features to view and download applied student data, improving placement coordination, transparency, and overall efficiency.
Authentication
Secure Admin and Student login using role-based authentication
Protected routes for sensitive operations

Admin Module
Add and manage company details
Maintain previous year recruitment rounds
Store and manage interview questions for student reference
View, download, and manage applied student lists
Generate reports to improve placement coordination

Student Module
Student registration and login
Resume upload functionality
Register and apply for upcoming placement drives
Track application status in real-time
